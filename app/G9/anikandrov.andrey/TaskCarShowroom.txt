Производитель автомобилей ведет статистику по своим диллерским центрам, записывая в файл информацию в следующем формате:
Дата; название дц; название автомобиля;комплектация;наценка дц
Найти следующую информацию:
  Для каждого автомобиля найти дилерский центр с максимальной наценкой 
  Найти дц, который продали больше всего различных комплектаций(автомобиль+комплектация), за последние 3 года
  Для каждого дц найти суммарный заработок 
В задаче должны использоваться элементы функционального программирования
Задача должна быть покрыта тестами с помощью JUnit
Должно использоваться логгирование
