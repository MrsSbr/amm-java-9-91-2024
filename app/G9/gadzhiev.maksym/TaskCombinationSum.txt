Combination Sum

Дан массив уникальных целых чисел-кандидатов и некоторое целое число,
вернуть список всех уникальных комбинаций, в которых сумма выбранных чисел равна этому числу.
Вы можете возвращать комбинации в любом порядке.

Одно и то же число-кандидат может быть использовано неограниченное количество раз.
Две комбинации уникальны, если частота хотя бы одного из выбранных чисел различна.

Пример 1:

Input: candidates = [2,3,6,7], target = 7
Output: [[2,2,3],[7]]

Пример 2:

Input: candidates = [2,3,5], target = 8
Output: [[2,2,2,2],[2,3,3],[3,5]]

Пример 3:

Input: candidates = [2], target = 1
Output: []

Ограничения:

1 <= candidates.length <= 30
1 <= candidates[i] <= 200
Все элементы candidates уникальны
1 <= target <= 500
