Contains Duplicate

Дан целочисленный массив nums, вернуть true, если какое-либо значение встречается в массиве не менее двух раз,
и вернуть false, если каждый элемент различен

Пример 1:

Input: nums = [1,2,3,1]
Output: true

Пример 2:

Input: nums = [1,2,3,4]
Output: false

Пример 3:

Input: nums = [1,1,1,3,3,4,3,2,4,2]
Output: true
 

Ограничения:

1 <= nums.length <= 10^5
-10^9 <= nums[i] <= 10^9
