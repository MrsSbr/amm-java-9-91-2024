Министр электроэнергетики анализирует работу различных типов электростанций, записывая
дату, тип электростанции, количество МВатт/часов. Всего известно о 17435 таких записях
(подумайте, как можно смоделировать ввод с помощью случайных чисел)

Необходимо найти следующую информацию:
    список типов электростанций, которые хотя бы один раз произвели больше 50 МВт/ч за последний месяц
    среднюю мощность для каждого типа электростанций за последние 3 месяца
    суммарную производимую мощность за последний год (для всех типов)

В задаче должны использоваться коллекции (за исключением Map), необходимо сравнить производительность
различных коллекций для этой задачи, выбрать наиболее подходящую (подходящие), аргументировать свой выбор

В задаче должны использоваться элементы функционального программирования
Задача должна быть покрыта тестами с помощью JUnit
