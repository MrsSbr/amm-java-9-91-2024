    Coin Change

Дан целочисленный массив монет, представляющий собой монеты разного номинала,
и целое число, представляющее общую сумму денег.

Верните наименьшее количество монет, которое вам нужно, чтобы разменять эту сумму.
Если эту сумму денег нельзя представить ни одной комбинацией монет, верните -1.

Можно предположить, что у вас есть бесконечное количество монет каждого вида.

Пример 1:

Input: coins = [1,2,5], amount = 11
Output: 3
Объяснение: 11 = 5 + 5 + 1

Пример 2:

Input: coins = [2], amount = 3
Output: -1

Пример 3:

Input: coins = [1], amount = 0
Output: 0
 

Ограничения:

1 <= coins.length <= 12
1 <= coins[i] <= 2^31 - 1
0 <= amount <= 10^4
