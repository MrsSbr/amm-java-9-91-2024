В таксопарке работают 20 шоферов, которые каждый день заправляют свои автомобили.
С целью контроля и сбора статистики в таксопарке решили на протяжении некоторого периода подсчитывать ежедневный расход
бензина.

Напишите класс, который принимает с клавиатуры количество рабочих дней в выбранный для анализа период (подумайте,
как можно смоделировать ввод с помощью случайных чисел). Для каждого рабочего дня программа должна прочитать расход
бензина каждого из 20 шоферов и вывести на экран общий расход бензина в этот день.

Кроме того, класс должен вывести на экран общий расход бензина за весь период - по каждому шоферу в отдельности.

В задаче должны использоваться коллекции (за исключением Map),
необходимо выбрать наиболее подходящую (подходящие) коллекцию для этой задачи, аргументировать свой выбор

В задаче должны использоваться элементы функционального программирования
Задача должна быть покрыта тестами с помощью JUnit
