Jump Game

Дан массив неотрицательных целых чисел nums, вы находитесь в его начале

Каждый элемент массива представляет собой максимальную длину прыжка из этой позиции.

Ваша цель - достичь последнего индекса за минимальное количество прыжков.

Можно предположить, что всегда можно добраться до последнего индекса.

Пример 1:

Input: nums = [2,3,1,1,4]
Output: 2
Объяснение: 1 прыжок: 0 -> 1, 2 прыжок 1 -> 4.

Пример 2:

Input: nums = [2,3,0,1,4]
Output: 2


Ограничения:

1 <= nums.length <= 10^4
0 <= nums[i] <= 1000
