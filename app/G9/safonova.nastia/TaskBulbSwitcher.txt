Bulb Switcher

Есть n лампочек. Изначально все они выключены.
Вы сначала включаете все лампочки, затем выключаете каждую вторую лампочку.

На третьем шаге вы переключаете каждую третью лампочку (включается, если она выключена, или выключается, если она включена).
В i-м раунде вы переключаете каждую i-ю лампочку.  В n-м раунде вы переключаете только последнюю лампочку.

Необходимо вернуть количество лампочек, горящих после n раундов.

Пример 1:
Input: n = 3
Output: 1
Объяснение: Изначально [off, off, off].
После первого раунда [on, on, on].
После второго раунда [on, off, on].
И после третьего раунда [on, off, off].

Пример 2:

Input: n = 0
Output: 0

Пример 3:

Input: n = 1
Output: 1

Ограничения:

0 <= n <= 10^9
