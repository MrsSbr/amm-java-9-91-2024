Distribute Candies

У Алисы есть n конфет, где каждая уникальная цифра представляет отдельный тип конфет.
Алиса заметила, что начала набирать вес, поэтому обратилась к врачу.

Врач посоветовал Алисе есть только n/2 конфет из тех, которые у нее есть (n всегда четно).
Но Алисе очень нравятся конфеты, и она хочет есть как можно больше разных конфет,пока следует советам доктора.

Учитывая целочисленный массив candyType длины n, вернуть максимальное количество различных типов конфет,
которые она может съесть, если она съест только n / 2 из них.

Пример 1:

Input: candyType = [1,1,2,2,3,3]
Output: 3
Объяснение: Алиса можеть съесть 6 / 2 = 3 конфет. У нее только 3 типа конфет,
поэтому она может съесть конфету каждого типа.

Пример 2:

Input: candyType = [1,1,2,3]
Output: 2
Объяснение: Алиса можеть съесть 4 / 2 = 2 конфет. Независимо от набора конфет [1,2], [1,3], or [2,3],
она может съесть только 2 типа.

Пример 3:

Input: candyType = [6,6,6,6]
Output: 1
Объяснение: Алиса можеть съесть 4 / 2 = 2 конфет. Несмотря на то, что она может съесть 2 конфеты,
у нее есть конфеты только одного типа.
 

Ограничения:

n == candyType.length
2 <= n <= 10^4
n - четное.
-10^5 <= candyType[i] <= 10^5
