На пасеке есть 35 ульев, которые каждый год приносят различное количество меда.
Для контроля и сбора статистики на пасеке решили подсчитывать ежегодный объем меда.

Напишите класс, который принимает с клавиатуры количество лет в выбранный для анализа период. Для каждого года программа
должна прочитать объем меда для каждого из 35 ульев (подумайте, как можно смоделировать ввод с помощью случайных чисел)
и вывести на экран статистику по объемам меда за каждый год.

Кроме того, класс должен вывести на экран общий объем меда за весь период по каждому улью.

В задаче должны использоваться коллекции (за исключением Map),
необходимо выбрать наиболее подходящую (подходящие) коллекцию для этой задачи, аргументировать свой выбор
В задаче должны использоваться элементы функционального программирования
Задача должна быть покрыта тестами с помощью JUnit
