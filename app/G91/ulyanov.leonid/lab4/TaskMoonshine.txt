Самогонщик Аркадий ведет учет выработанных алкогольных напитков, для этого записывает в файл следующую ссылку:
Дата; название напитка; список ингредиентов; объем; время настаивания
Найти: 
  Среднее время настаивания для каждого ингредиента
  Месяц в котором использовалось максимальное количество ингредиентов 
  Суммарный объем для каждой настойки
В задаче должны использоваться элементы функционального программирования
Задача должна быть покрыта тестами с помощью JUnit
Должно использоваться логгирование
